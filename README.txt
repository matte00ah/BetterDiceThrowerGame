# BetterDiceThrowerGame
A simple dice thrower for tabletop sessions with friends

To download, move to "master" branch and download the .zip file, then extract and have fun.

In game, to access the main menu, press the "esc" button: from there choose the type of dices you want to spawn and the amount, then clic apply. If you want to remove all dices in scene clic the corresponding button in the menu, same goes for the "exit application" button to close the game.
